## Overview

E-Bikolpo is a product boycott and alternative suggestion platform. It helps users find alternatives to products they wish to avoid. Built with **React**, **Firebase**, and **MongoDB**, this application aims to empower consumers through informed choices. 

## Features

- **User Authentication**: Secure login and registration using Firebase Authentication.
- **Product Suggestions**: Find alternatives to products you want to boycott.
- **Community Contributions**: Users can suggest new alternatives.
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.
- **Real-time Database**: Uses Firebase for real-time data updates.
- **User-friendly Interface**: Built with React and styled with Tailwind CSS and DaisyUI.

## Technologies Used

E-Bikolpo leverages a variety of technologies to provide a smooth user experience. Here are the main technologies used in this project:

- **React**: For building the user interface.
- **Firebase**: For backend services including authentication and hosting.
- **MongoDB**: For storing user and product data.
- **Node.js**: For server-side logic.
- **Express**: For building the web server.
- **JWT Token**: For secure user sessions.
- **DaisyUI**: For UI components.
- **Tailwind CSS**: For styling.
- **React Router DOM**: For navigation.
- **Vercel**: For deployment.

## Installation

To get started with E-Bikolpo, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/vijay2732004/E-Bikolpo-Client.git
   ```
   
2. Navigate into the project directory:
   ```bash
   cd E-Bikolpo-Client
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Set up your Firebase project and MongoDB Atlas. Update the configuration files with your credentials.

5. Start the development server:
   ```bash
   npm start
   ```

## Usage

Once you have the application running, you can access it in your web browser at `http://localhost:3000`. 

- **Sign Up/Login**: Create an account or log in to access features.
- **Search for Alternatives**: Use the search bar to find products you want to boycott.
- **Suggest Alternatives**: Contribute by suggesting alternatives to existing products.
